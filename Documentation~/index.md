# VeilMail Unity SDK Documentation

Welcome to the VeilMail Unity SDK documentation. This package provides full access to the Veil Mail API from Unity 2021.3 and later.

## Getting Started

For installation instructions, quick start guides, and API reference, visit the full documentation:

**[https://veilmail.xyz/docs/sdk-unity](https://veilmail.xyz/docs/sdk-unity)**

## Samples

Import samples via Window > Package Manager > VeilMail SDK > Samples:

- **Basic Email** -- Minimal email sending example using async/await
- **Template Email** -- Sending emails with templates and dynamic data
- **Server Proxy** -- Secure proxy pattern for shipped builds (recommended for production)
